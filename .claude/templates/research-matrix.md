## Usage Notes

**When to use this template**:
- Starting a new research initiative
- Making significant product decisions
- Needing to validate assumptions
- Building evidence for stakeholder alignment

**How to fill out**:
1. Start with the research objective - be clear on what decision this informs
2. List 3-5 key questions you need to answer
3. Select appropriate sources and methods
4. Create realistic timeline based on method complexity
5. Evaluate source quality using the matrix
6. Define clear success criteria
7. Document resources needed
8. Identify and mitigate risks
9. List expected deliverables
10. Get stakeholder approval before proceeding

**Tips**:
- Involve stakeholders early in planning
- Choose methods appropriate to questions
- Balance speed with confidence needs
- Plan for triangulation (multiple sources)
- Build in buffer time for unexpected issues
- Consider both qualitative and quantitative
- Document assumptions and constraints
