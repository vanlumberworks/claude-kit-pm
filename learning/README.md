# Learning Modules

Interactive learning resources for building technical literacy as a Product Manager.

## Available Courses

### JSON Fundamentals (`./json-exercises/`)
Learn to read and write JSON, essential for API work.
- Start with: `/learn-json` slash command
- Practice exercises included
- Real-world examples from product APIs

### API Basics (`./api-tutorials/`)
Understand APIs and how to test them without coding.
- Start with: `/learn-api` slash command
- Hands-on Postman tutorials
- API specification writing practice

### Debug Cases (`./debug-cases/`)
Learn systematic debugging for non-technical PMs.
- Real bug scenarios
- Practice investigations
- Bug report writing

## Learning Path

```
Week 1-2: JSON Fundamentals
├── Lesson 1: JSON basics
├── Lesson 2: Reading complex JSON
├── Lesson 3: Writing JSON
├── Lesson 4: Common patterns
└── ✓ Complete exercises

Week 3-4: API Basics
├── Module 1: What is an API?
├── Module 2: HTTP methods & status codes
├── Module 3: Authentication
├── Module 4: Testing with Postman
├── Module 5: Writing API specs
└── ✓ Test real product APIs

Week 5-6: Applied Skills
├── Debug real issues
├── Write API specs for PRDs
├── Create test cases
└── ✓ Certification

## Quick Start

```bash
# Start JSON course
/learn-json

# Start API course
/learn-api

# Practice with exercises
cd learning/json-exercises
open exercise-1.md

# Test your knowledge
cd learning/api-tutorials
open tutorial-1.md
```

## Certification

Complete all modules and pass assessments to earn:
- **JSON Literate Badge** - Can read/write JSON confidently
- **API Capable Badge** - Can test and document APIs
- **Technical PM Badge** - Full technical literacy for PMs
